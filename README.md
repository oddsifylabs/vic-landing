# VIC Landing Page

Static landing page for **VIC - Vegas Intelligence Console**.

Real-time signal intelligence for serious bettors. Deploy in seconds with Railway.

## Features

- 🚀 One-click Railway deployment
- 📱 Fully responsive design
- 🎨 Modern dark theme
- ⚡ Zero dependencies (HTML/CSS/JS)
- 🔗 Direct links to VIC, GitHub, and social channels

## Quick Start

### Local Development

```bash
npm install
npm run dev
```

Visit `http://localhost:3000`

### Deploy to Railway

Click the button below:

[![Deploy on Railway](https://railway.app/button.svg)](https://railway.app/new/template?template=https://github.com/oddsifylabs/vic-landing)

Or use your referral code:
```
https://railway.com/deploy/vegasintelligenceconsole?referralCode=3WX7iH
```
